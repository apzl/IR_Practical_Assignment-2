# IR_Practical_Assignment-2

Practical Assignment PR02 — Document Handling, Stop Words, Basic Boolean Retrieval.

## Setup


### 1. Create the environment

Using conda (Python 3.10):

```bash
conda create -n ir_pr02 python=3.10 -y
conda activate ir_pr02
```


### 2. Install requirements

```bash
pip install -r requirements.txt
```

## Running the tests

From the repository root:

```bash
python -m pytest public_tests/ -v
```

## Running the UI

```bash
python main.py
```
